Install:
    1.

        sudo apt update
        sudo apt upgrade

        sudo apt install hashcat

        (sudo apt install hashcat-nvidia)

    2.
        	
        wget https://hashcat.net/files/hashcat-6.2.5.7z

        unpack

        cd .../hashcat-6.2.5/

-----------------------------------------------------------------------------------------

Для видеокарт NVIDIA требуется «Драйвер NVIDIA» (440.64 или более поздняя версия) и «CUDA Toolkit» (9.0 или более поздняя версия)
