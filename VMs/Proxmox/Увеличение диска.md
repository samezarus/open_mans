# Увеличение размера диска виртуальной машины

- В веб-интерфейсе Proxmox → выбрать ВМ → Hardware → Hard Disk → Resize → ввести новое значение

- Зайти на виртуалку через SSH

- Зайти под `sudo`

- Посмотрите текущие диски

``` bash
lsblk
```

- Увидим что-то вроде

```
NAME                 MAJ:MIN RM  SIZE RO TYPE MOUNTPOINTS
sda                    8:0    0   47G  0 disk 
├─sda1                 8:1    0    1M  0 part 
├─sda2                 8:2    0    2G  0 part /boot
└─sda3                 8:3    0   45G  0 part 
  └─ubuntu--vg-lv--0 252:0    0   30G  0 lvm  /
sr0                   11:0    1  2.6G  0 rom  
```

- Увеличиваем размер раздела sda3 до конца диска

``` bash
apt update
apt install -y cloud-guest-utils
```

``` bash
growpart /dev/sda 3
```

- Расширям физический том LVM

``` bash
pvresize /dev/sda3
```

- Расширяем логический том LV

``` bash
lvextend -l +100%FREE /dev/ubuntu-vg/lv-0
```

```
Узнать путь логического тома LV

lvdisplay
```

- Расширить файловую систему

``` bash
resize2fs /dev/ubuntu-vg/lv-0
```
