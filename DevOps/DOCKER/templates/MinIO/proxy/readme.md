# MinIO + nginx + ssl

## С самоподписанными сертификатами

- Создать DNS-имя на локальном DNS-сервере
- На сервере развёртки создать самоподписанные сертификаты

## С letsencrypt сертификатами

- Приобрести DNS-имя у провайдера
- Выпустить сертификаты через letsencrypt

## Общие настройки

- MINIO_SERVER_URL - указвть DNS-имя сервера
- MINIO_SERVER_PORT - задать поррт для доступак api и консоли
