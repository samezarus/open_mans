# Настройка и запуск Sentry

## Инициализация

- Сгенирировать ключ

``` bash
make key
```

- Полученый ключ поместить в .env-файл

```
...
SENTRY_SECRET_KEY='<key>'
...
```

- Инициализация и Sentry (только при первом запуске проекта)

``` bash
make init
```
